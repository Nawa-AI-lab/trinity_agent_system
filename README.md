# Trinity AI Agent System

نظام ثلاثي الوكلاء الذكية المتخصصة في مجالات مختلفة:

1. **المهندس المعماري العودي (Ouroboros)** - متخصص في هندسة البرمجيات وتحسينها
2. **الرئيس التنفيذي المجهري (Micro-CEO)** - متخصص في إدارة الأعمال واتخاذ القرارات
3. **المركّب المعرفي الشامل (Polymath)** - متخصص في البحث وربط المعلومات

## المميزات

- ✅ هيكل معياري قابل للتوسيع
- ✅ تكامل مع OpenAI و Anthropic
- ✅ أتمتة المتصفح باستخدام Playwright
- ✅ إدارة المهام والأدوات
- ✅ واجهة REST API كاملة
- ✅ توثيق شامل

## البدء السريع

```bash
# استنساخ المشروع
git clone https://github.com/YOUR_USERNAME/trinity_agent_system.git
cd trinity_agent_system

# إنشاء البيئة الافتراضية
python3 -m venv venv
source venv/bin/activate

# تثبيت المكتبات
pip install -r requirements.txt

# تثبيت Playwright
playwright install chromium

# تشغيل النظام
python -m src.main
```
